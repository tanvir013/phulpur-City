# Phulpur Upazila Service Directory

A simple landing page for a local service directory website in Phulpur Upazila, Mymensingh, Bangladesh.

## How to Run

1. Open `index.html` in your web browser.
2. The page is responsive and uses Bootstrap for styling.

## Features

- About section with information on Phulpur Upazila
- Services section with categories like Healthcare, Education, Shopping, etc.
- Contact information
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
